## 🦖 What?

Ralph Caveman = Smart loop + Terse speak.
- **Ralph**: Iterative dev loop. No stop until done.
- **Caveman**: Drop fluff. Save 75% tokens. Fast. Cheap.

## 🛠️ Setup

1. Have Gemini CLI.
2. Put skill in `.gemini/skills/ralph-caveman/`.
3. Start loop.

## 🏃 Run

```bash
/ralph-caveman "build me world"
```
